# ProjekUASPBO
UAS PBO 24SI2
Cara Memainkan
1. Memilih Mode
Saat permainan dimulai, Anda akan dihadapkan pada dua pilihan:
Lawan Bot: Anda bermain melawan komputer yang memilih gerakan secara acak.
Lawan Teman: Mode multiplayer lokal. Player 1 memilih gerakan terlebih dahulu (pilihan akan disembunyikan), kemudian Player 2 memilih, baru hasil akhirnya ditampilkan.

3. Aturan Dasar & Energi
Setiap pemain memulai dengan Skor 0 dan Energi Penuh (10/10).
Setiap ronde yang selesai akan memulihkan +1 Energi untuk Player 1.
Jika Energi tidak cukup, pemain tidak bisa memilih jurus spesial.

3. Daftar Gerakan (Moveset)
Berikut adalah daftar gerakan yang bisa dipilih beserta biaya energi dan efeknya:
Gerakan Dasar (Batu / Kertas / Gunting)
Biaya: 0 Energi
Efek: Peraturan standar (Batu kalahkan Gunting, Gunting kalahkan Kertas, Kertas kalahkan Batu).

Batu Raksasa
Biaya: 6 Energi
Cooldown: 3 Giliran
Efek: Sangat kuat. Gerakan ini dapat mengalahkan Gunting DAN Kertas sekaligus.

Kertas Seribu
Biaya: 4 Energi
Efek: Pengganda Skor. Jika Anda menang menggunakan jurus ini, Anda mendapatkan 2x Poin.
Gunting Kembar
Biaya: 3 Energi
Efek: Pertahanan. Jika Anda kalah saat menggunakan jurus ini, status kalah dianulir menjadi "Seri" sehingga lawan tidak mendapat poin.

4. Penggunaan Power-Ups
Di panel Inventory, terdapat tombol bantuan yang bisa diaktifkan:
Shield: Mengaktifkan pelindung untuk menahan kekalahan pada ronde berikutnya.
Clairvoyance: Menambah +5 Energi secara instan.
Teknologi yang Digunakan
HTML5: Struktur halaman.
CSS3: Styling antarmuka, layout responsif, dan animasi CSS.

JavaScript (ES6+): Logika permainan menggunakan Class dan Object-Oriented Programming (OOP).

Cara Menjalankan
Pastikan Anda memiliki file index.html (atau nama file HTML Anda).
Buka file tersebut menggunakan web browser modern (Google Chrome, Firefox, Edge, atau Safari).
